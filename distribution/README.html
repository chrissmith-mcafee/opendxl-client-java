Data Exchange Layer (DXL) Java SDK

The DXL Java SDK contains the following directories:

  • doc
    • Documentation for the DXL Java SDK
  • lib
    • DXL Java libraries
  • sample
    • Samples that demonstrate use of the DXL Java SDK
